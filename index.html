HTML

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem padrão usada para criar páginas web e estruturar o conteúdo que vemos online.

CSS

CSS significa Folhas de Estilo em Cascata (do inglês, Cascading Style Sheets). É uma linguagem utilizada para descrever a apresentação visual de documentos HTML e XML.

JAVASCRIPT

JavaScript é uma linguagem de programação de alto nível, conhecida por sua flexibilidade e ampla utilização em desenvolvimento web para adicionar interatividade e dinamismo a páginas e aplicações.
